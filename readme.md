# Aula de Formulário

Um projeto educacional sobre criação de formulários HTML com validação.

## 📋 Descrição

Este projeto foi desenvolvido durante aulas de desenvolvimento web, focando em:
- Estrutura de formulários HTML
- Validação de dados
- Estilização com CSS
- Interatividade com JavaScript

## 🎯 Objetivo

Aprender os conceitos fundamentais de formulários web e boas práticas de UX.
